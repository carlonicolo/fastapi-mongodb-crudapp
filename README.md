# Simple CRUD FastAPI and MongoDB app

All you need is:
- Python
- MongoDB
- Pycharm Community version or another IDE

Then you can install this python libraries:
```pip install pymongo fastapi uvicorn```  

Now start the FastAPI:  
```uvicorn index:app --reload```
